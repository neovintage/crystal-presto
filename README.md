# presto.cr

prestodb crystal driver

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     presto.cr:
       github: neovintage/presto.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "presto.cr"
```
